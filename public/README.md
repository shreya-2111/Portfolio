# Public Folder

This folder contains static assets that will be served directly.

## Add Your Resume Here

1. Save your resume as a PDF file
2. Name it `resume.pdf`
3. Place it in this `public` folder

Your resume will then be accessible at `/resume.pdf` and the "Download Resume" button will work.

## Current Structure

```
public/
├── README.md (this file)
└── resume.pdf (add your resume here)
```

## Note

- Any file you place in this folder will be publicly accessible
- The folder contents are copied to the build output during deployment
- You can also add other static files like images, icons, etc.
